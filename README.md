# Feihong's PureScript Quickstart

## Prerequisites

```
brew install node
npm install -g yarn
```

## Create project

```
mkdir purescript-quickstart && cd purescript-quickstart
yarn add --dev purescript spago
yarn spago init
```

## Commands

Compile and run `src/Main.purs`

    yarn spago run

Compile and run `test/Main.purs`

    yarn spago test
